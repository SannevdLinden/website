API Documentation

An API is available for those who want to request the todo's of this site to their own website. To go to the API click this link or through sannespock.pythonanywhere.com/snippets/ . Here you can find an overview of all the todo's. If you are logged in you can also add a todo through the user interface on the bottom. You can request different data types, for example the json data can be retrieved through http://sannespock.pythonanywhere.com/snippets/?format=json to get all the todo's. To get specific todo's you can click on the url in the overview or browse to http://sannespock.pythonanywhere.com//website/snippets/id/ , you should replace id with the id number of that specific todo. In the user interface you can also edit the specific todo here, if you are logged in. If you go to http://sannespock.pythonanywhere.com/users/ all the users are shown and if you go to http://sannespock.pythonanywhere.com/schema/ you can get the corejson data by clicking the get button or through http://sannespock.pythonanywhere.com/schema/?format=corejson .
